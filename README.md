## Astur

The RFID-inventory-app of De Havik.

Astur is a genus containing hawks ([wikipedia](https://en.wikipedia.org/wiki/Astur_(bird)))

## Set up (local development)

1. Node and npm needs to be installed (hint: use nvm to manage multiple node versions).
2. Install the CLI from ionic globally via `npm install -g @ionic/cli`
3. Clone the repo.
4. Run `ionic serve` from the root directory.

## Primary goal of the app?
Integrate with the Chainway C72 handheld RFID scanner.
